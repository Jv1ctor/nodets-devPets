# nodets-devPets
Projeto feito no módulo do curso de Node + typescript da b7web

### Pré-requisitos globais:
`npm i -g nodemon typescript ts-node`

### Instalação
`npm install` 

### Para rodar o projeto
`npm run start-dev`
